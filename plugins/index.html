TTY Plugins

Below you will find listing of all the plugins from the tty ecosystem.

Each plugin can be installed independently, therefore you can mix and match what your CLI app needs. However, by installing tty gem, you will include all the plugins.

Plugin name Description
pastel Terminal strings styling with intuitive and clean API.
tty-color Terminal color capabilities detection.
tty-cursor Move terminal cursor around.
tty-pager Terminal output paging in a cross-platform way.
tty-platform Detecting different operating systems.
tty-progressbar A flexible progress bars drawing in terminal emulators.
tty-prompt A robust command line prompt.
tty-screen Terminal screen properties detection.
tty-spinner A terminal spinner for tasks with non-deterministic time.
tty-table A flexible and intuitive table output generator.
tty-which Platform independent implementation of Unix which command.
